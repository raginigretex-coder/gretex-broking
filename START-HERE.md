# 🚀 Deploy Gretex Financial to Vercel - Quick Start

## ⚡ Fastest Way to Deploy (3 Steps)

### Step 1: Install Vercel CLI
Open PowerShell or Command Prompt and run:
```bash
npm install -g vercel
```

### Step 2: Login to Vercel
```bash
vercel login
```
This will open your browser to login.

### Step 3: Deploy!
Navigate to your project folder and run:
```bash
cd C:\Users\Gretex\OneDrive\Desktop\GretexShare
vercel --prod
```

**That's it!** Your website will be live in minutes! 🎉

## 🆘 Troubleshooting

**Problem:** `vercel: command not found`
**Solution:** Install Node.js first from https://nodejs.org, then install Vercel CLI

**Problem:** Deployment fails
**Solution:** Check the error message and ensure all files are present

**Problem:** Pages show 404
**Solution:** The `vercel.json` file handles routing - make sure it's in the root directory
